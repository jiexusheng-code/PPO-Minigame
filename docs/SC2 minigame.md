| **场景名称 (Scenario Name)**    | **核心目标 (Core Goal)**                      | **场景描述与配置**                                           | **主要特点 / 考察能力 (Key Features)**                       |
| ------------------------------- | --------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **MoveToBeacon**                | **移动基础** 控制单位到达指定地点             | 地图上会随机出现一个信标（Beacon），玩家需控制一名陆战队员（Marine）走到信标处，触碰后信标重置位置。 | 考察Agent最基础的空间感知和移动指令执行能力，不涉及战斗或规划。 |
| **CollectMineralShards**        | **多体路径规划** 控制两个单位收集碎片         | 地图上散落着多个晶体矿碎片，玩家控制两名陆战队员，需要在限时内收集尽可能多的碎片。 | **贪婪策略 vs 全局最优**。考察多智能体协作（避免走重复路线）和路径优化效率。 |
| **FindAndDefeatZerglings**      | **探索与索敌** 在迷雾中寻找并消灭敌人         | 地图被战争迷雾覆盖，敌人（跳虫 Zergling）随机分布。玩家控制三名陆战队员探索地图并击杀它们。 | **部分可观测性 (POMDP)**。 考察在信息不完全情况下的地图探索策略、视野管理以及遇敌后的反应。 |
| **DefeatRoaches**               | **战斗微操 (Micro)** 以少胜多/通过操作获胜    | 玩家控制一队陆战队员对抗一队蟑螂（Roach）。蟑螂数值更强，正面硬刚必输。 | **集火与走A (Kiting)**。 考察“甩枪”技巧、集火攻击特定目标以及保持距离的战斗微操能力。 |
| **DefeatZerglingsAndBanelings** | **高级微操 (Advanced Micro)** 规避AOE伤害     | 玩家控制陆战队员对抗跳虫和毒爆虫（Baneling）。毒爆虫自爆会造成范围伤害。 | **散兵操作 (Splitting)**。 考察高难度的微操，必须通过分散站位（散枪兵）来最小化AOE伤害，并优先处理高威胁目标。 |
| **CollectMineralsAndGas**       | **经济运营 (Macro)** 优化资源采集效率         | 玩家拥有一个基地和初始SCV，目标是在限时内采集尽可能多的晶体矿和高能瓦斯。 | **资源调度**。 考察工人的生产、分配（采矿 vs 采气）以及由于基地饱和度带来的边际效益管理。 |
| **BuildMarines**                | **生产流程 (Build Order)** 建立产线并生产单位 | 目标是制造尽可能多的陆战队员。需要采集资源 -> 造补给站 -> 造兵营 -> 产兵。 | **动作序列与决策**。 考察对科技树的理解、资源分配（经济 vs 军事）以及长序列动作的规划能力。 |
